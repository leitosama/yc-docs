В [консоли управления]({{link-console-main}}) на странице с информацией о сервисном аккаунте для каждого ключа отображаются дата и время его последнего использования. Эта информация позволяет отслеживать случаи несанкционированного использования ключей, а также удалять неиспользуемые ключи без риска нарушить работу сервисов {{ yandex-cloud }}.
